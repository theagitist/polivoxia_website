# The Ground Remains - Website

This is a mini website for *The Ground Remains*, a print-and-play game inspired by *Indigenomicon* by Jodi A. Byrd.

## Files

- `index.html` - Main website page with all sections
- `styles.css` - Styling for the website
- `THE-GROUND-REMAINS.pdf` - Original PDF zine (8 pages)
- `the_ground_remains_map_bg.png` - Game board image

## Website Sections

1. **Designer's Statement** - Introduction and design philosophy
2. **Game Rules** - How to play the game
3. **Game Board** - Visual board with printing instructions
4. **Theoretical Documentation** - Academic context and further reading

## Updating Content

The website currently contains placeholder content based on the game's description. To update with specific content from the PDF:

1. Review `THE-GROUND-REMAINS.pdf` for exact text
2. Update the relevant sections in `index.html`
3. Ensure all links and references are accurate

## Styling

The website uses a clean, modern design with:
- Responsive layout for mobile and desktop
- Accessible color scheme
- Print-friendly styles
- Smooth scrolling navigation
