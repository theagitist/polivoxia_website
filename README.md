# POLIVOXIA.COM

Portfolio website showcasing random projects by Adri M.

## Overview

This is a single-page portfolio website featuring an interactive gallery of projects. The site displays various creative works including interactive web projects, writing, art, music, documentaries, and games.

## Structure

- `index.html` - Main portfolio page with gallery
- `img/` - Image assets including logos and project previews
- `costarica/` - Subdirectory for Costa Rica project
- `stories_in_whispers/` - Interactive poetry project
- `downloads/` - Downloadable resources

## Features

- Responsive design with separate desktop and mobile galleries
- Interactive card-based gallery with hover effects and animations
- Parallax effects on mouse movement
- Custom typography using TypeKit fonts (bmx-radical, courier-std, Lato)
- Instagram integration

## Projects Featured

1. **Stories in Whispers** - Interactive poetry creation platform
2. **Transphobia in Costa Rica** - Human rights documentation
3. **Journeyways** - Board game about becoming
4. **TransRants** - Writing blog (rants, poetry, essays)
5. **Sujeto del Inconsciente** - Video dance ensemble with live visuals
6. **Ounces - Stemmata** - Concert visuals for live shows
7. **Programa Convivir** - Documentary series about rural high school communities

## Technologies

- HTML5
- CSS3 (with animations and responsive design)
- JavaScript (for parallax effects)
- External services for screenshot generation (WordPress mshots, Thum.io, Screenshot Machine)

## Browser Support

Modern browsers with support for:
- CSS Grid
- Flexbox
- CSS animations
- JavaScript ES6+
