# POLIVOXIA.CA

Portfolio site for Adri M. (theagitist): liminal spaces, interactive art, writing, games, and creative projects.

Live: <https://polivoxia.ca>

## Structure

- **/** — Homepage with project cards (The Ground Remains, Stories in Whispers, Costa Rica, Journeyways, TransRants, Sujeto del Inconsciente, Ounces, Programa Convivir)
- **/the_ground_remains/** — Print-and-play game/zine inspired by *Indigenomicon* by Jodi A. Byrd. Tabs: Statement, Rules, Board, Downloads. PDFs: full zine, colonial grid only.
- **/costarica/** — Costa Rica section
- **/downloads/** — Main downloads index
- **/sitemap.xml** — Sitemap

## The Ground Remains assets

- `the_ground_remains/downloads/the_ground_remains.pdf` — Full zine/game
- `the_ground_remains/downloads/colonial_grid.pdf` — Game board only
- `the_ground_remains/img/colonial_grid.jpg` — Board image (for BOARD tab and display)

## Tech

Static HTML/CSS/JS. Typekit (Adobe Fonts) for Nunito and others. Cookie consent and Google Analytics (opt-in).
